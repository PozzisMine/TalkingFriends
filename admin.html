Excel (.xlsx) → data.json для сайта статистики

1. Загрузить Excel

В Excel должны быть столбцы: date, channel, channelId, subscribers, views, videos

2. Предпросмотр данных

date channel channelId subscribers views videos

3. Как загрузить на сайт

После скачивания файла data.json загрузи его в GitHub:

Add file → Upload files → data.json → Commit changes

Потом подожди 1–2 минуты и обнови сайт.
