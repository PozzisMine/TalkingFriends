# Mixerno Live Addon

Файлы:
- mixerno.html — отдельная live-страница
- mixerno.css — стиль страницы
- mixerno.js — получение данных Mixerno и график

## Куда добавить

Загрузи эти 3 файла в корень GitHub-репозитория:

index.html
style.css
script.js
data.json
mixerno.html
mixerno.css
mixerno.js

## Как открыть

https://твой-ник.github.io/название-репозитория/mixerno.html

## Важно

Эта страница НЕ записывает данные в data.json.
Она только показывает live-данные из Mixerno.
