# Сайт истории версий игр

Полностью статический сайт для GitHub Pages.

## Как использовать

1. Создай новый репозиторий на GitHub, например `game_versions_site`.
2. Загрузи все файлы из этого архива в репозиторий.
3. В настройках GitHub Pages включи публикацию из ветки `main` (или `master`).
4. Через несколько минут сайт будет доступен по адресу:
   https://<твой_логин>.github.io/game_versions_site/

## Как редактировать

- Все данные об играх находятся в `data/games.json`.
- Добавляй новые игры, изменяй версии, описания и иконки (по URL).
- Никакой backend не нужен — всё работает в браузере.
